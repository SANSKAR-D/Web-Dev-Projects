\begin{problem}{Love percentage II}{standard input}{standard output}{1 second}{256 megabytes}

You are given three inputs: two name strings (name1, name2) and a specific word (word) which will be one of ["love", "like", "hate"]. The goal is to calculate a normalized compatibility score between the two names based on a specific algorithm.

Here are the step-by-step requirements:

1. Character Removal: Identify letters that appear in both name1 and name2, and remove them from both strings.

2. String Concatenation: Create two new strings:

    str1 = name1 + word + "true" + name2

    str2 = name1 + word + "false" + name2

   (Note: Use the modified name1 and name2 from Step 1).

3. ASCII Conversion: Convert every character in str1 and str2 into its numerical ASCII value and combine them into one long string of digits.

4. Digit Reduction: For both digit strings, repeat the following process until only a 2-digit number remains:

    Take the first digit and the last digit of the string.

    Add them together.

    Remove the first and last digits from the string.

    Append the sum to the beginning of the remaining string.

5. This yields ans1 (from str1) and ans2 (from str2).

Normalization: Convert ans1 and ans2 into a scale where their sum exactly equals 100. Return these two values.


\Note
1 <= name1.length(), name2.length() <= 1000

word ∈ {"love", "like", "hate"}

Strings consist of lowercase English letters only.

\end{problem}
